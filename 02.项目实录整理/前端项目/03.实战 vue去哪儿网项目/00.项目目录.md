src目录 项目源代码
    main.js: 项目的入口文件
    App.vue: 项目最原始的根组件
    router文件夹: index.js存放项目的路由
    components文件夹: 项目的可复用小组件
    asset文件夹:  图片资源

旧版本vue cli有config文件夹 存放dev和product的配置信息
旧版本vue cli有build文件夹 存放webpack的打包配置信息 不需要我们做改动


旧版本2.8.2 创建新vue项目时 vue init webpack newproject

运行项目 npm run dev
