# 历史和简介
1998年在w3c接管css后发布了css2，随即发布css2.1修复了一些bug。

2011年开始css被分为多个模块单独升级，统称css3，这些模块是：
css 选择器
css 媒体查询
css color

css英文官方文档是 css specification
