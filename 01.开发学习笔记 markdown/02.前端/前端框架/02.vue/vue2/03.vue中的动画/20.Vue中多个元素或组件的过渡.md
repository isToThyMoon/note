# 多个元素的过渡：

```html
<head>
    <style>
        .fade-enter, .fade-leave-to {
            opacity: 0;
        }
        .fade-enter-active, .fade-leave-active {
            transition: opacity 2s;
        }
    </style>
</head>

<body>
    <div id="root">
        <transition 
        name="fade"
        mode="in-out"
        >
            <div v-if="show" key="hello">hello world</div>
            <div v-else key="new">not world</div>
        </transition>

        <button @click="handleClick">切换</button>
    </div>

    <script>
        var vm = Vue({
            el: '#root',
            data: {
                show: true
            },
            method:{
                handleClick: function() {
                    this.show = ! this.show
                }
            }
        })

    </script>
</body>
```
只是像之前单个div时的那要写，if 和else切换时，dom并没有动画过渡的效果，因为vue中会尽量复用dom，节省性能，所以这里并不会生效。
解决方法是只需要给每个div不同的key值。

transition的mode属性设置了多个元素切换时是先展示动画后切换dom，还是先切换dom后展示动画。
因为同时生效的进入和离开的过渡不能满足所有要求，所以 Vue 提供了过渡模式
in-out：新元素先进行过渡，完成之后当前元素过渡离开。
out-in：当前元素先进行过渡，完成之后新元素过渡进入。

# 多组件的过渡
可以基础组件来写，
也可以通过动态组件实现。

```html
<head>
    <style>
        .fade-enter, .fade-leave-to {
            opacity: 0;
        }
        .fade-enter-active, .fade-leave-active {
            transition: opacity 2s;
        }
    </style>
</head>

<body>
    <div id="root">
        <transition 
        name="fade"
        mode="in-out"
        >
            <child-one v-if="show"></child-one>
            <child-two v-else></child-two>
            <component :is="type">
        </transition>

        <button @click="handleClick">切换</button>
    </div>

    <script>
        Vue.component('child-one', {
            template: '<div>child-one</div>'
        })
        Vue.component('child-two', {
            template: '<div>child-two</div>'
        })
        var vm = Vue({
            el: '#root',
            data: {
                show: true,
                type: child-one
            },
            method:{
                handleClick: function() {
                    this.show = ! this.show;
                    this.type = this.type === 'child-one' ? 'child-two' : 'child-one';
                }
            }
        })

    </script>
</body>
```
