# directives 指令
React 使用两个指令（directive）向 与 React 服务器组件兼容的捆绑器（bundler） 提供指令（instruction）。
这些指令仅在你 使用 React 服务器组件（RSC） 或构建可适配库时需要。

'use client' 标记组件在客户端上执行的源文件。

'use server' 标记可以从客户端代码调用的服务器函数。在异步函数的最顶部添加 'use server' 以标记该函数可以被客户端执行。
