# 扩展符号

```JavaScript
...
... 叫扩展符号, 作用是把数组解开成单独的元素
var a1 = [1, 2, 3]
var a2 = [...a1, 4]
// 结果是 [1, 2, 3, 4]

...可以用来传参
var a1 = [1, 2, 3]
log(...a1)
// 相当于 log(1, 2, 3)
// 类似于 log.apply(window, a1)
```

# 解包 解构赋值
解包是其他语言(Python) 中学过来的特性, 见下方例子
    
    var [a, b] = [1, 2]
    // a b 分别被赋值为 1 2
    // 相当于下面的老代码
    var arr = [1, 2]
    var a = arr[0]
    var b = arr[1]

使用这个语法可以一行代码给多个变量赋值
所以也就可以一行代码交换变量值
` [a, b] = [b, a] `
