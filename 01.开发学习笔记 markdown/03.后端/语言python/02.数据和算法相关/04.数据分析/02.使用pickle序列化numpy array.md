序列化到磁盘
```py
import pcickle
import numpy as np

x = np.arrange(10)
f = open('x.pkl', 'wb')

# 序列化
pickle.dump(x, f)

# 反序列化
f = open('x.pkl', 'rb')

pickle.load(f)
```
